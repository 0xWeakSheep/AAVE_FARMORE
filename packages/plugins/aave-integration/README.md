# aave-integration

Core AAVE V3 integration and transaction execution.

## Overview

This plugin provides AI-powered assistance for interacting with the AAVE V3 protocol, including:

- Supplying assets to AAVE
- Borrowing against collateral
- Repaying loans
- Withdrawing supplied assets
- Executing transactions

## Installation

```bash
/plugin install aave-integration
```

## Usage

Use this plugin when you need to:

- Execute transactions on AAVE V3
- Interact with AAVE smart contracts
- Manage lending positions

## Skills

- `aave-integration` - Core AAVE integration skill
