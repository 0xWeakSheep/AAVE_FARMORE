# aave-planner

Position planning and strategy optimization for AAVE V3.

## Overview

This plugin provides AI-powered assistance for planning AAVE V3 positions, including:

- APY calculations and quotes
- Position strategy planning
- Supply/borrow optimization
- Simulation of positions

## Installation

```bash
/plugin install aave-planner
```

## Usage

Use this plugin when you need to:

- Plan AAVE positions
- Calculate APYs
- Optimize lending strategies
- Simulate positions before execution

## Skills

- `aave-planner` - AAVE position planning skill
