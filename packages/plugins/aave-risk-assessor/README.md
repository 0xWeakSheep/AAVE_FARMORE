# aave-risk-assessor

Risk assessment and health factor monitoring for AAVE positions.

## Overview

This plugin provides AI-powered assistance for assessing risks in AAVE V3 positions, including:

- Health factor monitoring
- Liquidation risk assessment
- Collateral ratio analysis
- Risk warnings and alerts

## Installation

```bash
/plugin install aave-risk-assessor
```

## Usage

Use this plugin when you need to:

- Assess position risk
- Monitor health factors
- Analyze liquidation risks
- Evaluate collateral safety

## Skills

- `aave-risk-assessor` - AAVE risk assessment skill
